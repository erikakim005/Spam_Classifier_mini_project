# 📩 Spam Message Classifier

## 📌 Project Overview  
This project is a simple machine learning model that classifies text messages as spam or not spam (ham).  
It is built to practice basic NLP (Natural Language Processing) and understand how text data can be converted into numerical features for model training.

## ⚙️ Technologies Used  
- Python  
- pandas (data handling)  
- scikit-learn (machine learning)  

## 🧠 How It Works  
1. Load text data (messages with labels: spam or ham)  
2. Convert text into numerical features using CountVectorizer  
3. Train a classification model (Naive Bayes)  
4. Predict whether a new message is spam or not  

## 🚀 Features  
- Classifies input messages  
- Simple and fast model  
- Beginner-friendly implementation
  
## 📊 Example  
Input:  
> "Free money now!!!"  
Output:  
> Spam
